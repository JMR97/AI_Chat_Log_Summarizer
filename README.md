# AI_Chat_Log_Summarizer


A Python tool to summarize chat logs between a User and an AI.


##Features
- Message counts
- Top 5 keywords
- Simple NLP with NLTK



How to Run
```bash
  python AI_Chat_Log_Summarizer.py
```
